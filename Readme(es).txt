Este es mi primer proyecto con una board Arduino Uno y en total me tomó tres meses (algunas pausas) hasta su terminación.
 Este prototipo tiene ya 4 meses de prueba con resultados satisfactorios.
Vivo en un país donde no hay tiendas de componentes electrónicos ni de partes mecánicas.
Exceptuando la board Arduino en todo el proyecto se emplearon partes de equipos descontinuados.

El proyecto se dividió en tres partes fundamentales:

1- Diseño y montaje de la placa electrónica.

2- Creación y puesta a punto del software de control.

3- Ensamblaje de todo el sistema mecánico.


En la etapa final se procedió a la unión de las partes y la calibración de todo el sistema.

Consulte el manual para una información mas detallada.
